# [PRD] Daily K-Wisdom (가칭: 하루의 지혜) 기획서

## 1. 개요 (Overview)

- **프로젝트명:** Daily K-Wisdom (매일 만나는 한국의 지혜)
- **목적:** 외국인을 대상으로 한국의 속담, 명언, 고전 문구 등을 매일 하나씩 소개하여 한국 문화와 언어에 대한 관심을 충족시킴.
- **타겟 유저:** K-POP, K-Drama 등을 통해 한국 문화에 관심이 생긴 외국인, 한국어 학습자.
- **플랫폼:** 모바일 웹 (Mobile Web) - 반응형 웹사이트

## 2. 핵심 기능 (Core Features)

### 2.1. 일일 콘텐츠 (Daily Quote)

- **로직:** 매일 자정(00:00)을 기준으로 새로운 문구가 노출되어야 함.
- **데이터 구성:**
  - `id`: 고유 번호
  - `korean`: 한글 원문 (예: "가는 말이 고와야 오는 말이 곱다")
  - `english`: 영어 번역 (예: "Nice words for nice words")
  - `meaning`: (선택) 속담의 내포된 의미 설명
- **데이터 소스:** 로컬 JSON 파일 또는 하드코딩된 배열 (MVP 단계)

### 2.2. 오디오 기능 (TTS)

- **기능:** 스피커 아이콘 클릭 시 해당 한글 문구를 읽어줌.
- **기술:** Web Speech API (`window.speechSynthesis`) 활용.
- **설정:** 언어(`lang`)는 'ko-KR'로 고정.

### 2.3. UI/UX (Mobile First)

- **레이아웃:** 원페이지(One-page) 세로 스크롤 없는 고정형 디자인 (100vh).
- **구성:**

  1.  **Header:** 서비스 로고/타이틀 (작고 심플하게)
  2.  **Main:** 화면 중앙에 카드 형태로 문구 배치
  3.  **Action:** 문구 옆 또는 하단에 '스피커(듣기)' 버튼 배치

- 모바일 위주 디자인 가로 길이 최대 800px 하고 가운데 정렬로 하기

## 3. 디자인 & 스타일 가이드 (Design & Style)

### 3.1. 컨셉 (Concept)

- **키워드:** Minimalist, Zen, Traditional, Paper-like
- **배경:** 순백색보다는 약간의 미색(Warm White, #F9F9F9) 또는 한지 느낌의 텍스처 사용 권장.

### 3.2. 타이포그래피 (Typography) - **중요**

'명언'과 '지혜'라는 주제에 맞게 가독성이 좋으면서도 진중한 느낌을 주는 **세리프(Serif, 명조체)** 계열을 추천합니다. 구글 폰트(Google Fonts)를 사용합니다.

- **한글 폰트 (Korean):**
  - **1순위: Nanum Myeongjo (나눔명조)** - 가장 클래식하고 기품 있음.
  - **2순위: Gowun Batang (고운바탕)** - 조금 더 부드럽고 서정적인 느낌.
  - **3순위: Song Myung (송명)** - 붓글씨 느낌이 강해 전통적인 느낌을 줌.
- **영문 폰트 (English):**
  - **1순위: Lora** - 나눔명조와 가장 잘 어울리는 우아한 세리프 폰트.
  - **2순위: EB Garamond** - 클래식한 서적 느낌.

### 3.3. 컬러 팔레트 (Color Palette)

- **Text (Main):** #333333 (완전한 검정보다 부드러운 다크 그레이)
- **Text (Sub):** #666666 (영어 번역용)
- **Accent:** #C62828 (한국적인 붉은색 - 낙관 느낌) 또는 #1A237E (차분한 남색) - 포인트 아이콘용.

## 4. 기술 스택 (Tech Stack)

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS 권장 - 가볍게 제작)
- **Deployment:** GitHub Pages 또는 Vercel (무료 호스팅)
- **TTS:** Browser Built-in Web Speech API (별도 라이브러리 설치 불필요)
